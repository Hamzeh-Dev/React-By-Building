# 5 Shopping List

A simple app built to **practice managing arrays in state**: adding, deleting, toggling, and editing items dynamically.

## Goal
Learn how to work with **state arrays**:
- Add items to an array in state
- Delete items from the array
- Toggle a boolean property in array items
- Edit an itemâ€™s value
- Understand how React re-renders components when state changes

## How to Run

1. Clone the repo:
```bash
git clone git@github.com:Hamzeh-Dev/React-By-Building.git
```

2. Navigate to the project:
```bash
cd '.\Level 1\5-shopping-list'
```

3. Install dependencies:
```bash
npm install
```

4. Start the app:
```bash
npm run dev
```

Open in browser at http://localhost:3000
