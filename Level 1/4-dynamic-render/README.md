# 4 Dynamic Render

A simple app to **understand dynamic rendering, loading states, and side effects** using `useEffect`.

## Goal
Learn how React dynamically renders content:  
- Show a **loading skeleton** while fetching data.  
- Automatically fetch data when the component mounts using `useEffect`.  
- Update the UI when state changes without refreshing the page.  
- Understand **Virtual DOM** efficiency in updating only what changes.

## How to Run

1. Clone the repo:
```bash
git clone git@github.com:Hamzeh-Dev/React-By-Building.git
```

2. Navigate to the project:
```bash
cd '.\Level 1\4-dynamic-render'
```

3. Install dependencies:
```bash
npm install
```

4. Start the app:
```bash
npm run dev
```

Open in browser at http://localhost:3000
