# 2 Reusable Card Component

A simple app built to **understand reusable components** and how to pass **dynamic props** to a child component.

## Goal
Learn how to create reusable components that display data received via props, allow the parent to control the state, and update dynamically when the parent state changes.

## How to Run

1. Clone the repo:
```bash
git clone git@github.com:Hamzeh-Dev/React-By-Building.git
```

2. Navigate to the project:
```bash
cd '.\Level 1\2-reusable-card-component'
```

3. Install dependencies:
```bash
npm install
```

4. Start the app:
```bash
npm run dev
```

Open in browser at http://localhost:3000
