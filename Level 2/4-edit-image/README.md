# Image Editor

A small interactive project that allows users to upload an image, apply filters and transformations, and export the result for preview. This project helps you understand state management, controlled inputs, dynamic styles, and canvas manipulation.

### What You Will Learn

- Handling file uploads in React

- Using useState to manage multiple states (filters, transforms)

- Using useRef to access DOM elements

- Applying CSS filters dynamically

- Applying transformations: rotate, flip, scale

- Exporting an image using the HTML Canvas API

## How to Run

1. Clone the repo:

```bash
git clone git@github.com:Hamzeh-Dev/React-By-Building.git
```

2. Navigate to the project:

```bash
cd level-2/4-edit-image
```

3. Install dependencies:

```bash
npm install
```

4. Start the app:

```bash
npm run dev
```

Open in browser at http://localhost:3000

### Challenge

- Try to add additional filters like hue-rotate or opacity
